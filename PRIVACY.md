# Privacy Policy

QuickAskAI does not collect, store, or transmit any personal information.

This extension only makes API calls to AI services (GitHub Copilot or OpenAI-compatible endpoints) based on user queries.

User credentials are stored securely in Windows Credential Manager and are never shared with third parties.

## Data Storage

- Provider configurations are stored locally at `%USERPROFILE%\Documents\QuickAskAI\providers.json`
- Chat history is stored locally at `%USERPROFILE%\Documents\QuickAskAI\conversations.json`
- GitHub Copilot tokens are stored in Windows Credential Manager only

## Third-Party Services

This extension communicates with:
- GitHub Copilot API (api.github.com/copilot)
- OpenAI-compatible API endpoints configured by the user

No user data is shared with any third parties beyond the AI services explicitly chosen by the user.
